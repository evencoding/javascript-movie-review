## 어플리케이션 실행 방법

### 1. 패키지 다운받기

```bash
npm i
```

위 명령어로 필요한 패키지를 다운로드 해 주세요!

### 2. 프로젝트 실행

```bash
npm run start
```

위 명령어를 터미널에 입력해주세요!

### 3. 지금 가장 인기있는 영화를 확인해보세요!

```
로고를 클릭하는 것으로 인기 영화 페이지로 돌아갈 수 있어요!
검색창에 영화의 이름을 검색해보세요!
```
